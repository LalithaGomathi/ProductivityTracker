
Full Deluxe Agent Productivity Package
=====================================

Contents:
- streamlit_app/    (Streamlit UI)
- backend/          (FastAPI analysis service)
- connectors/       (ServiceNow and AWS Connect connector stubs)
- migrations/       (placeholder for Alembic migrations)
- init_db/          (Postgres init SQL)
- docker-compose.yml

Quick start (requires Docker):
1. docker-compose up --build
2. Streamlit UI: http://localhost:8501
3. Backend API: http://localhost:8000
